## 1. 소개 (Introduction)

### Image Compare Tool이란?

Image Compare Tool은 두 개의 이미지를 나란히 비교하고, 픽셀 단위의 차이를 시각적으로 분석할 수 있는 GUI 애플리케이션입니다.

### 주요 기능

| 기능 | 설명 |
|------|------|
| **이중 이미지 로드** | 두 개의 이미지를 독립적으로 불러오기 |
| **3-패널 뷰** | 원본 1, 원본 2, 차이 이미지를 동시에 표시 |
| **픽셀 차이 시각화** | 차이를 컬러 히트맵으로 표현 |
| **동기화 줌** | 세 패널이 동시에 줌 인/아웃 |
| **팬/드래그** | 이미지 내 자유로운 이동 |
| **크기/위치 정보** | 실시간 이미지 정보 표시 |

### 사용 사례

- 이미지 캘리브레이션 전/후 비교
- 이미지 처리 알고리즘 결과 검증
- 품질 검사 및 결함 탐지
- 버전 간 이미지 차이 분석

## 2. 설치 및 실행 (Installation & Launch)

### 방법 1: 실행 파일 사용 (권장)

Python 설치가 필요 없는 독립 실행 파일입니다.

```
1. dist\ImageCompareTool.exe 파일을 원하는 위치에 복사
2. ImageCompareTool.exe를 더블 클릭하여 실행
```

### 방법 2: Python 스크립트 실행

Python이 설치된 환경에서 소스 코드로 실행합니다.

```bash
# 1. 필수 패키지 설치
pip install numpy Pillow

# 2. 애플리케이션 실행
python src/image_compare_app.py
```

### 시스템 요구사항

| 항목 | 요구사항 |
|------|----------|
| 운영체제 | Windows 10/11 |
| 메모리 | 4GB RAM 이상 권장 |
| 디스플레이 | 1280x720 이상 해상도 |
| (Python 실행 시) | Python 3.8+ |

```
┌─────────────────────────────────────────────────────────────────┐
│  [Load Image 1] [Load Image 2] │ Zoom: [-] 100% [+] │ [Center] [Fit to Window]  │  ← 컨트롤 바
├─────────────────────┬─────────────────────┬─────────────────────┤
│      Image 1        │      Image 2        │     Difference      │
│  ┌───────────────┐  │  ┌───────────────┐  │  ┌───────────────┐  │
│  │Size: 256x256  │  │  │Size: 256x256  │  │  │Size: 256x256  │  │  ← 정보 표시
│  │View:(78,78)   │  │  │View:(78,78)   │  │  │View:(78,78)   │  │
│  │   100x100     │  │  │   100x100     │  │  │   100x100     │  │
│  │               │  │  │               │  │  │               │  │
│  │    [이미지]    │  │  │    [이미지]    │  │  │  [차이 이미지] │  │  ← 이미지 패널
│  │               │  │  │               │  │  │               │  │
│  └───────────────┘  │  └───────────────┘  │  └───────────────┘  │
├─────────────────────┴─────────────────────┴─────────────────────┤
│  Ready - Load two images to compare              Position: (x,y) │  ← 상태 바
└─────────────────────────────────────────────────────────────────┘
```

### 컨트롤 바 (Control Bar)

| 버튼 | 기능 |
|------|------|
| **Load Image 1** | 첫 번째 이미지 파일 불러오기 |
| **Load Image 2** | 두 번째 이미지 파일 불러오기 |
| **- / +** | 줌 아웃 / 줌 인 (10% 단위) |
| **Center** | 이미지 중심으로 이동, 100x100 뷰, 100% 줌 |
| **Fit to Window** | 전체 이미지가 보이도록 줌 조정 |

### 이미지 패널 (Image Panels)

- **Image 1 (왼쪽)**: 첫 번째 로드된 이미지
- **Image 2 (중앙)**: 두 번째 로드된 이미지
- **Difference (오른쪽)**: 두 이미지의 픽셀 차이

각 패널 상단에 표시되는 정보:
- **Size**: 이미지 전체 크기 (너비 x 높이)
- **View**: 현재 보이는 영역 (시작좌표) 크기

### 상태 바 (Status Bar)

- **왼쪽**: 현재 상태 메시지, 로드된 파일 경로
- **오른쪽**: 마우스 커서의 이미지 좌표

## 4. 기본 사용법 (Basic Usage)

### 4.1 이미지 불러오기

1. **Load Image 1** 버튼 클릭
2. 파일 선택 대화상자에서 첫 번째 이미지 선택
3. **Load Image 2** 버튼 클릭
4. 두 번째 이미지 선택
5. 자동으로 차이 이미지가 생성됨

**지원 포맷**: PNG, JPEG, TIFF (16-bit 포함), BMP, GIF

### 4.2 줌 (Zoom)

| 방법 | 동작 |
|------|------|
| **마우스 휠** | 위로 스크롤 = 줌 인, 아래로 = 줌 아웃 |
| **+ 버튼** | 10% 줌 인 |
| **- 버튼** | 10% 줌 아웃 |

줌 범위: 10% ~ 1000%

### 4.3 이동 (Pan)

1. 이미지 패널 위에서 마우스 왼쪽 버튼 클릭
2. 버튼을 누른 채로 드래그
3. 세 패널이 동시에 이동

### 4.4 중심으로 돌아가기

**Center** 버튼 클릭:
- 이미지 중심으로 이동
- 100% 줌으로 리셋
- 100x100 픽셀 영역 표시

## 5. 고급 기능 (Advanced Features)

### 5.1 차이 이미지 해석

차이 이미지는 컬러 히트맵으로 표시됩니다:

```
🔴 빨간색 (255, 0, 0)   = 최대 차이 (픽셀 값이 많이 다름)
🟢 녹색   (0, 255, 0)   = 차이 없음 (동일한 픽셀)
🟡 노란색/주황색        = 중간 정도의 차이
```

**계산 방식**:
```python
diff = abs(image1_grayscale - image2_grayscale)
normalized = diff / max(diff) * 255
```

### 5.2 16-bit TIFF 이미지 처리

16-bit TIFF 이미지는 자동으로 8-bit로 정규화되어 표시됩니다:
- 원본 데이터 범위를 0-255로 매핑
- 시각화 목적으로만 변환 (원본 데이터 보존)

### 5.3 크기가 다른 이미지 비교

두 이미지의 크기가 다른 경우:
- 경고 메시지가 표시됨
- 작은 쪽 크기에 맞춰 비교
- 좌상단 기준으로 정렬

### 5.4 좌표 추적

마우스를 이미지 위에 올리면:
- 상태 바에 현재 픽셀 좌표 표시
- 형식: `Position: (x, y)`
- 원본 이미지 기준 좌표

## 6. 튜토리얼 (Step-by-Step Tutorial)

### 튜토리얼 1: 기본 이미지 비교

**목표**: 두 테스트 이미지를 비교하고 차이점 확인하기

**단계**:

1. **애플리케이션 실행**
   ```
   ImageCompareTool.exe 더블클릭
   ```

2. **첫 번째 이미지 로드**
   - [Load Image 1] 클릭
   - `data\test_image_A.png` 선택
   - 왼쪽 패널에 이미지 표시됨

3. **두 번째 이미지 로드**
   - [Load Image 2] 클릭
   - `data\test_image_B.png` 선택
   - 중앙 패널에 이미지 표시됨
   - 오른쪽에 차이 이미지 자동 생성

4. **차이점 분석**
   - 오른쪽 Difference 패널 확인
   - 빨간 부분 = 차이가 큰 영역
   - 녹색 부분 = 동일한 영역

5. **상세 확인**
   - 마우스 휠로 줌 인
   - 드래그로 관심 영역 이동
   - [Center]로 원위치

---

### 튜토리얼 2: 캘리브레이션 전/후 비교

**목표**: 이미지 캘리브레이션 효과 분석

**단계**:

1. **이미지 로드**
   - Image 1: `data\W32_nocal.tif` (캘리브레이션 전)
   - Image 2: `data\W32_cal.tif` (캘리브레이션 후)

2. **전체 보기**
   - [Fit to Window] 클릭
   - 전체 이미지 개요 확인

3. **영역별 분석**
   - 차이가 큰 영역(빨간색) 줌 인
   - 해당 영역의 원본 이미지 비교

4. **정량적 확인**
   - 상태 바의 "Max diff" 값 확인
   - 값이 클수록 차이가 큼

---

### 튜토리얼 3: 위치 오프셋 검출

**목표**: 이미지 정렬 오류 확인

**단계**:

1. **테스트 이미지 로드**
   - Image 1: `data\test_circle_A.png`
   - Image 2: `data\test_circle_B.png` (10픽셀 오프셋)

2. **차이 패턴 분석**
   - 원형 테두리에 차이 발생
   - 한쪽은 빨강, 반대쪽은 녹색

3. **오프셋 방향 확인**
   - 빨간색 → Image 2에서 더 밝음
   - 이동 방향 추정 가능

## 7. 문제 해결 (Troubleshooting)

### 7.1 이미지가 검게/하얗게 보여요

**원인**: 16-bit 이미지의 값 범위 문제

**해결**:
- 자동 정규화가 적용되어야 함
- 이미지 파일이 손상되지 않았는지 확인
- 다른 이미지 뷰어에서 먼저 확인

### 7.2 "Size Mismatch" 경고가 나와요

**원인**: 두 이미지의 크기가 다름

**해결**:
- 비교 전 이미지 크기를 맞추기
- 또는 작은 크기로 비교 진행 (자동)

### 7.3 줌이 작동하지 않아요

**원인**: 이미지 패널에 포커스가 없음

**해결**:
- 이미지 패널 영역을 먼저 클릭
- +/- 버튼 사용

### 7.4 실행 파일이 실행되지 않아요

**원인**: Windows 보안 설정

**해결**:
1. 파일 우클릭 → 속성
2. "차단 해제" 체크
3. 적용 후 재실행

### 7.5 차이 이미지가 전부 녹색이에요

**원인**: 두 이미지가 완전히 동일함

**해결**:
- 정상 동작임
- 다른 이미지 쌍으로 테스트

## 8. FAQ

### Q1: 어떤 이미지 포맷을 지원하나요?

**A**: PNG, JPEG, TIFF (8-bit, 16-bit), BMP, GIF를 지원합니다.

### Q2: 최대 이미지 크기 제한이 있나요?

**A**: 소프트웨어적 제한은 없으나, 매우 큰 이미지(10000x10000 이상)는 메모리 사용량이 증가합니다.

### Q3: 컬러 이미지도 비교할 수 있나요?

**A**: 예. 단, 차이 계산은 그레이스케일로 변환 후 수행됩니다.

### Q4: 차이 이미지를 저장할 수 있나요?

**A**: 현재 버전에서는 지원하지 않습니다. 향후 업데이트 예정입니다.

### Q5: Mac이나 Linux에서도 사용 가능한가요?

**A**: Python 스크립트는 크로스 플랫폼 호환됩니다. 실행 파일(.exe)은 Windows 전용입니다.

### Q6: 3개 이상의 이미지를 비교할 수 있나요?

**A**: 현재는 2개 이미지 비교만 지원합니다.
